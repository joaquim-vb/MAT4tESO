\section*{SdA1. Potències i radicals. Operacions amb fraccions}

\begin{capsula}
Abans de començar, recorda la descomposició en factors primers: divideix sempre per nombres primers fins arribar a 1. Després, usa aquests factors per simplificar fraccions i escriure potències amb base primera.
\end{capsula}

\textbf{1. Descompon els següents nombres en factors primers:}
\begin{enumerate}[label=\alph*)]
    \item $588 =$ \vspace{0.7cm}
    \item $72 =$ \vspace{0.7cm}
    \item $196 =$ \vspace{0.7cm}
\end{enumerate}
\newpage
\begin{capsula}
Per simplificar fraccions amb descomposició en factors primers, factoritza numerador i denominador i després cancel·la els factors comuns.
\end{capsula}

\textbf{2. Simplifica al màxim aquestes fraccions utilitzant la descomposició en factors primers de l’exercici anterior quan sigui possible:}

\begin{enumerate}[label=\alph*)]
    \item $\dfrac{588}{72} =$ \vspace{1cm}
    \item $\dfrac{72}{196} =$ \vspace{1cm}
    \item $\dfrac{301}{50} =$ \vspace{1cm}
\end{enumerate}

\begin{capsula}
Quan multipliques o divides potències amb la mateixa base, suma o resta exponents. Si la base és negativa o hi ha potències dins d'altres potències, revisa bé l'ordre d'aplicació.
\end{capsula}

\textbf{3. Aplica les propietats de les potències i calcula:}
\begin{enumerate}[label=\alph*)]
    \item $(-2)^{-1}\cdot(-2)^{-3}\cdot(-2)^4 =$ \vspace{0.8cm}
    \item $4^0 =$ \vspace{0.8cm}
    \item $\dfrac{5^5}{5^2} =$ \vspace{0.8cm}
    \item $(7^{-3})^2 =$ \vspace{0.8cm}
\end{enumerate}

\newpage
\begin{capsula}
Per escriure una potència amb exponent positiu, fes servir que $a^{-n}=\dfrac{1}{a^n}$. També pots reescriure expressions com $\left(\dfrac{a}{b}\right)^{-n}=\left(\dfrac{b}{a}\right)^n$.
\end{capsula}

\textbf{4. Expressa les següents potències amb l’exponent positiu:}
\begin{enumerate}[label=\alph*)]
    \item $3^{-3} =$ \vspace{0.8cm}
    \item $5^{-2} =$ \vspace{0.8cm}
    \item $2^3\cdot 3^{-2} =$ \vspace{0.8cm}
    \item $5^6\cdot 7^{-3} =$ \vspace{0.8cm}
\end{enumerate}

\begin{capsula}
Per passar un nombre a producte de potències de base primera, primer factoritza cada nombre i després eleva cada factor al seu exponent.
\end{capsula}

\textbf{5. Expressa el resultat en forma de producte de potències de base primera.}
\begin{enumerate}[label=\alph*)]
    \item $50^3 =$ \vspace{0.8cm}
    \item $12^2\cdot 30^3 =$ \vspace{1cm}
\end{enumerate}

\begin{capsula}
En notació científica, escriu el nombre com $a\cdot 10^n$ amb $1\le a<10$.
\end{capsula}

\textbf{6. Expressa els següents nombres en notació científica:}
\begin{enumerate}[label=\alph*)]
    \item $5.000.000.000 =$ \vspace{0.8cm}
    \item $82.400.000.000.000.000 =$ \vspace{0.8cm}
\end{enumerate}

\begin{capsula}
Quan extreguis factors d'una arrel, busca quadrats o cubs perfectes dins del radicand. Si no pots simplificar més, deixa l'arrel com està.
\end{capsula}

\textbf{7. Extreu factors de l’arrel quan sigui possible:}
\begin{enumerate}[label=\alph*)]
    \item $\sqrt{81} =$ \vspace{0.8cm}
    \item $\sqrt{18} =$ \vspace{0.8cm}
\end{enumerate}

\begin{capsula}
Per unir factors sota una sola arrel, pensa que $a\sqrt{b}=\sqrt{a^2\cdot b}$ i que les arrels amb el mateix índex es poden combinar.
\end{capsula}

\textbf{8. Expressa com una sola arrel:}
\begin{enumerate}[label=\alph*)]
    \item $2\cdot \sqrt{5} =$ \vspace{0.8cm}
    \item $6\cdot \sqrt{x} =$ \vspace{0.8cm}
    \item $13\cdot \sqrt{54} =$ \vspace{0.8cm}
\end{enumerate}

\begin{capsula}
Per fer operacions amb fraccions, recorda: si sumes o restes, busca denominador comú; si multipliques, multiplica numeradors i denominadors; si divides, multiplica per la inversa.
\end{capsula}

\textbf{9. Fes les operacions amb fraccions següents:}
\begin{enumerate}[label=\alph*)]
    \item $\dfrac{1}{3}\cdot \dfrac{1}{3}+\dfrac{3}{4} =$ \vspace{1cm}
    \item $\left(\dfrac{2}{6}-3\right)\div \left(\dfrac{2}{3}-\dfrac{1}{5}\right) =$ \vspace{1.2cm}
\end{enumerate}

\section*{SdA2. Polinomis}

\begin{capsula}
Recorda que un polinomi s'escriu com a suma de termes de la forma $a_n x^n$. El grau és el màxim exponent amb coeficient diferent de zero i el terme independent és el que no té $x$.
\end{capsula}

\textbf{1. Donat el polinomi } $P(x)=3x^4-2x^2-6x+1$ \textbf{, indica:}
\begin{enumerate}[label=\alph*)]
    \item El grau del polinomi: \dotfill\vspace{0.4cm}
    \item El terme de grau 2: \dotfill\vspace{0.4cm}
    \item El coeficient del terme de grau 1: \dotfill\vspace{0.4cm}
    \item El coeficient del terme de grau 3: \dotfill\vspace{0.4cm}
    \item El valor numèric de $P(2):$ \dotfill\vspace{0.6cm}
    \item El valor numèric de $P(-2):$ \dotfill\vspace{0.6cm}
    \item El terme independent: \dotfill\vspace{0.4cm}
\end{enumerate}

\begin{capsula}
Per avaluar un polinomi a un valor concret, substitueix $x$ pel nombre i calcula pas a pas respectant la jerarquia d'operacions.

Per agrupar termes, suma els coeficients de cada potència de $x$. Ordena el resultat amb el terme de major grau al principi.
\end{capsula}

\textbf{2. Escriu els següents polinomis agrupant els termes del mateix grau i ordenant-los segons els seus exponents:}
\begin{enumerate}[label=\alph*)]
    \item $x^3 - 2x^2 - x^4 + 3x^3 + 4x^4 - 3x + 15 - x - 5 =$ \vspace{1cm}
    \item $x^5 - 6x^5 - 4x^2 + 2 + 4x^3 - 10 + 10x^5 =$ \vspace{1cm}
\end{enumerate}

\newpage



\textbf{3. Realitza les següents sumes i restes de polinomis. Agrupa termes semblants i ordena segons l'exponent:}
\begin{enumerate}[label=\alph*)]
    \item $(3x^3 + 5x^2 + 1) + (8x^3 + 4x^2 + 5) =$ \vspace{1.2cm}
    \item $(-2x^3 - 4x^2 + 2x + 1) + (x^3 - 6x^2 - 5x + 2) =$ \vspace{1.2cm}
    \item $(-2x^8 - 3x^7 + 5x^3 - 1) - (5x^8 + 6x^7 + x^3 - x^2 + 6x) =$ \vspace{1.2cm}
    \item $(6x^4 + 5x^3 - 2x^2 - 3) - (x^4 + 5x^3 + 2x^2 - 1) =$ \vspace{1.2cm}
\end{enumerate}

\begin{capsula}
Quan restis polinomis, canvia signe a cada terme del segon polinomi i després suma terme a terme.
\end{capsula}

\textbf{4. Realitza els següents productes. Agrupa el resultat:}
\begin{enumerate}[label=\alph*)]
    \item $2x^2\cdot(3x-2) =$ \vspace{0.8cm}
    \item $(x+1)\cdot(x+2) =$ \vspace{0.8cm}
    \item $(2x+3)\cdot(2x-3) =$ \vspace{0.8cm}
    \item $(5x^2+1)\cdot(3x^3-5x) =$ \vspace{1cm}
\end{enumerate}

\begin{capsula}
Aplica la propietat distributiva ($a(b+c)=ab+ac$) i identifica termes semblants per agrupar-los.
\end{capsula}

\textbf{5. Transforma les següents equacions de segon grau fins obtenir la forma simplificada $ax^2+bx+c=0$:}
\begin{enumerate}[label=\alph*)]
    \item $3x^2+6x-45 = 2x - 30$ \vspace{1cm}
    \item $3x - 2x^2 = 8x - 5x^2 + 10$ \vspace{1cm}
\end{enumerate}

\begin{capsula}
Passos útils: passa tots els termes al costat esquerre, ordena per graus i simplifica coeficients.
\end{capsula}

\textbf{6. Resol les següents equacions de segon grau:}
\begin{enumerate}[label=\alph*)]
    \item $x^2 - 5x + 6 = 0$ \vspace{1cm}
    \item $3x^2 + 6x - 45 = 0$ \vspace{1cm}
\end{enumerate}

\begin{capsula}
Recorda la fórmula general $x=\dfrac{-b\pm\sqrt{b^2-4ac}}{2a}$ i comprova el discriminant $\Delta=b^2-4ac$ abans de resoldre.
\end{capsula}

\section*{SdA3. Equacions}

\begin{capsula}
En una equació, l'objectiu és deixar la incògnita sola. Si hi ha parèntesis, primer desenvolupa. Si hi ha fraccions, intenta eliminar-les multiplicant per un denominador comú.
\end{capsula}

\textbf{1. Donat el polinomi $P(x)=3x^4-2x^2-6x+1$, indica:}
\begin{enumerate}[label=\alph*)]
    \item El grau del polinomi: \dotfill\vspace{0.4cm}
    \item El terme de grau 2: \dotfill\vspace{0.4cm}
    \item El coeficient del terme de grau 1: \dotfill\vspace{0.4cm}
    \item El coeficient del terme de grau 3: \dotfill\vspace{0.4cm}
    \item El valor numèric de $P(2)$: \dotfill\vspace{0.6cm}
    \item El valor numèric de $P(-2)$: \dotfill\vspace{0.6cm}
    \item El terme independent: \dotfill\vspace{0.4cm}
\end{enumerate}

\begin{capsula}
Abans de combinar polinomis, agrupa els termes del mateix grau. Recorda que el signe menys davant d'un parèntesi afecta tots els termes de dins.
\end{capsula}

\textbf{2. Escriu els següents polinomis agrupant els termes del mateix grau i ordenant-los segons els seus exponents:}
\begin{enumerate}[label=\alph*)]
    \item $x^3-2x^2-x^4+3x^3+4x^4-3x+15-x-5 =$ \vspace{1cm}
    \item $x^5-6x^5-4x^2+2+4x^3-10+10x^5 =$ \vspace{1cm}
\end{enumerate}

\begin{capsula}
Quan sumes o restes polinomis, suma o resta els coeficients dels termes semblants.
\end{capsula}

\textbf{3. Realitza les següents sumes i restes. El resultat l’has de donar agrupant els termes semblants i ordenant els termes segons els seus exponents:}
\begin{enumerate}[label=\alph*)]
    \item $(3x^3+5x^2+1)+(8x^3+4x^2+5) =$ \vspace{1cm}
    \item $(-2x^3-4x^2+2x+1)+(x^3-6x^2-5x+2) =$ \vspace{1cm}
    \item $(-2x^8-3x^7+5x^3-1)-(5x^8+6x^7+x^3-x^2+6x) =$ \vspace{1cm}
    \item $(6x^4+5x^3-2x^2-3)-(x^4+5x^3+2x^2-1) =$ \vspace{1cm}
\end{enumerate}

\begin{capsula}
En un producte de polinomis, aplica la distributiva. Si veus una identitat notable, fes-la servir directament.
\end{capsula}

\textbf{4. Realitza els següents productes. El resultat l’has de donar agrupant els termes semblants i ordenant els termes segons els seus exponents:}
\begin{enumerate}[label=\alph*)]
    \item $2x^2\cdot(3x-2) =$ \vspace{0.8cm}
    \item $(x+1)\cdot(x+2) =$ \vspace{0.8cm}
    \item $(2x+3)\cdot(2x-3) =$ \vspace{0.8cm}
    \item $(5x^2+1)\cdot(3x^3-5x) =$ \vspace{1cm}
\end{enumerate}

\begin{capsula}
Per transformar una equació a la forma $ax^2+bx+c=0$, passa tots els termes a un mateix costat i simplifica.
\end{capsula}

\textbf{5. Transforma les següents equacions de segon grau fins obtenir la forma simplificada $ax^2+bx+c=0$:}
\begin{enumerate}[label=\alph*)]
    \item $3x^2+6x-45=2x-30$ \vspace{1cm}
    \item $3x-2x^2=8x-5x^2+10$ \vspace{1cm}
\end{enumerate}

\begin{capsula}
Per resoldre una equació de segon grau, pots factoritzar si és possible o utilitzar la fórmula general. Comprova sempre el discriminant.
\end{capsula}

\textbf{6. Resol les següents equacions de 2n grau:}
\begin{enumerate}[label=\alph*)]
    \item $x^2-5x+6=0$ \vspace{1cm}
    \item $3x^2+6x-45=0$ \vspace{1cm}
\end{enumerate}
